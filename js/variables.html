Variables

No es necesario declarar variables en Javascript.

Se pueden usar directamente: 

    a = 25;
    b = 33;

Se puede usar la palabra reservada var para declarar variables. 

    var conta = 21;
    var factor = 10;

¿Cuál es la diferencia? El ámbito

Sin var cualquier variable es global.
Con var se declaran variables locales que serán visibles sólo en el ámbito donde son declaradas.
